# Markdown Formatting in Geode

We have a YouTube tutorial you may want to watch which may help you use these tags (and GD's default ones)!
<YouTube videoId="rmvGedzp_yc" title="part 7" />

Geode includes several classes for rendering Markdown in Geometry Dash:

- `MDPopup`
- `MDTextArea`

These classes allow you to use most Markdown features. If you're new to Markdown or want a refresher, check out the <Link href="https://www.markdownguide.org/">Markdown Guide</Link>.

In addition to standard Markdown formatting, Geode also supports several custom tags and integrations.

## Standard GD Tags

Geode Markdown classes allow you to also use RobTop's **color tags**, making your text much fancier. For a guide on how to use these color tags, read this <Link href="">guide</Link> or watch the video above!

## Displaying Mods

To display a mod neatly in Markdown (for example in `about.md`), use the following syntax:

```markdown
<mod:geode.node-ids>
```

Replace `geode.node-ids` with any valid mod ID.

## Linking Users

To create a clickable link to a user's profile, use the following syntax:

```markdown
[OmgRod](<user:17369066>)
```

Replace:

- `OmgRod` with the text you want displayed
- `17369066` with the user's **account ID**

<Alert type="info" title="Note:">
    This uses the user's **account ID**, not their user ID.
</Alert>

You can find a user's account ID by:

- Viewing their profile on <Link href="https://gdbrowser.com">GDBrowser</Link>
- Using the <Link href="https://geode-sdk.org/mods/cvolton.betterinfo">BetterInfo</Link> mod in-game

## Linking Levels

To create a clickable link to a level, use the following syntax:

```markdown
[Bloodbath](<level:10565740>)
```

Replace:

- `Bloodbath` with the text you want displayed
- `10565740` with the level ID

<Alert type="info" title="Note:">
    This uses the level ID shown in Geometry Dash and on sites such as GDBrowser.
</Alert>

You can find a level's ID by:

- Viewing the level in-game
- Opening the level page on <Link href="https://gdbrowser.com">GDBrowser</Link>

## Hex Color Tags

As you may know, Geometry Dash already has support in *some* places for color tags, but they are very limited in what colors you can have. Because of this, Geode now allows you to use hex color tags, which have the following syntax:

```markdown
    <c-2f7fe0>Hello, world!</c>
```

You may replace `Hello, world!` with any of your own text, and `2f7fe0` with any hex code you want.

Additionally, by combining these color tags, you can create cool gradients, like in BetterEdit's `about.md`:

```markdown
    <c-11f917>B</c><c-12f534>e</c><c-13f348>t</c><c-14f155>t</c><c-14ef67>e</c><c-15ed76>r</c>Edit
```
